Finds the optimal subset of pallets to maximize profit without exceeding the truck's capacity using dynamic programming.

This function constructs a DP table to evaluate all possible combinations of pallets and capacities, making sure the selected set has the highest possible profit while staying within the weight limit.

Time Complexity: O(i × j)
