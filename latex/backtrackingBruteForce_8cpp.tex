Solves the 0/1 Knapsack problem using backtracking. 

The asymptotic complexity of this algorithm is O(2\texorpdfstring{$^\wedge$}{\string^}n) in the worst case, where n is the number of pallets. However, the actual performance depends on the specific dataset and can be significantly improved with pruning techniques.
