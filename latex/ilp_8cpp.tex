Function that solves the 0/1 knapsack problem using Integer Linear Programming (ILP) with OR-\/\+Tools. 

This function uses the OR-\/\+Tools library to create a Mixed Integer Programming (MIP) solver. The function\textquotesingle{}s worst-\/case asymptotic complexity is O(2\texorpdfstring{$^\wedge$}{\string^}\+N), where n is the number of items (pallets) in the dataset, however, in pratice, the solver is efficient and can solve large instances of the problem.
